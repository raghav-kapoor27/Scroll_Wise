Scroll Wise

A browser extension designed to reduce accidental exposure to disturbing video content by blurring previews and prompting before playback.

üìå Installation Guide
1. Download the Extension Files

Download the ZIP folder of this project.

Extract it anywhere on your computer.

2. Install in Chrome (or any Chromium-based browser)

Open Chrome.

Go to chrome://extensions in the address bar.

Turn on Developer mode (top-right).

Click Load unpacked.

Select the extracted project folder (mindful-scroll-globalclick/).

The extension should now appear in your extensions list.

3. Install in Microsoft Edge

Open Edge.

Go to edge://extensions.

Enable Developer mode.

Click Load unpacked.

Pick your extracted folder.

Done.

4. (Optional) Install in Brave / Opera / Vivaldi

The steps are exactly the same:
